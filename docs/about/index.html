About Daily Random Leetcode Challenges

Welcome to Daily Random Leetcode Challenges

Mỗi ngày, chúng tôi sẽ chọn một challenge ngẫu nhiên từ Leetcode để bạn giải quyết.

Tham gia thử thách với quy ước như sau:

Convention

Submit code trên Leetcode.

Tạo thư mục lưu lời giải tương ứng với số thứ tự ngày.

Lời giải hợp lệ lưu trong thư mục tương ứng, đặt tên file theo cấu trúc: lcnumber_solver.extension, trong đó:

  • number: số thứ tự của challenge trên Leetcode.
  • solver: tên của người giải.
  • extension: phần mở rộng tương ứng với ngôn ngữ trong lời giải.
